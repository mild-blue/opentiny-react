# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `opentiny-react`, please report it
privately using GitHub's ["Report a vulnerability"](https://github.com/mild-blue/opentiny-react/security/advisories/new)
form rather than opening a public issue.

For vulnerabilities in the **editor itself**, please report them to the
[OpenTiny repository](https://github.com/mild-blue/opentiny/security).

We will acknowledge your report and work with you on a fix and disclosure timeline.
